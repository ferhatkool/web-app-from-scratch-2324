Kilian Valkhof - 07-02-2024

Op 7 febuari kwam Kilian Valkhof langs als gastspreker, waar hij voorbeelden ging uitleggen over JavaScript features die tegenwoordig ook gemakkelijk in CSS te maken zijn. Zo noemde hij voorbeelden als:

  • 'custom toggles', waarbij je de styling van een checkbox kan vervangen met een switch/slider
  • 'datalists', waarbij de browser real-time suggesties geeft tijdens het typen
  • 'In-page transitions', waarbij er op een klik element een scroll functie is gemaakt dat scrollt naar een gespecifiseerd stuk tekst
  • 'Carousels', die een element bewegend maken dat kan stoppen met scrollen en zichzelf vast zet in een bepaalde positie
  • 'Accordions', wat teksten gemakkelijk in-en uitklapbaar maakt.
  • 'dialogs', wat als het ware een moderne pop-up melding is dat compleet gestyled kan worden naar wens.
  • 'Container queries', waarmee a.d.h.v. een container size bepaald kan worden wat de size van een element is.

Enkele van deze voorbeelden heb ik toegepast op mijn eigen website. Het gaat om de Accordions en dialogs. Ik vond het erg handig om accordions te gebruiken, omdat ik onderdelen in mijn site had zitten die niet perse altijd relevant, maar wel vrij groot waren. Door de accordions heb ik mijn site overzichtelijker kunnen maken door deze onderdelen inklapbaar te maken. Ook heb ik gebruik gemaakt van dialogs, waarvoor ik eerst iets simpels met JavaScript heb gemaakt. Hierbij werden weergegeven elemenenten telkens vervangen voor andere elementen wanneer er op een knop werd gedrukt. Dat werkte wel gewoon goed, alleen met de dialogs kon ik echt mini-pagina's creeëren op mijn one-page website met minder JavaScript. De dialogs heb ik gestyled, zodat het meer lijkt op aparte pagina's dan alleen een pop-up element. Zelf heb ik bedacht om de dialogs te kunnen sluiten door buiten de dialogs om te klikken, wat voor mij een fijnere manier is om pop-ups te sluiten.

Als niet JavaScript professional vond ik het erg fijn om voorbeelden te zien van wat er allemaal mogelijk is met alleen HTML en CSS. Momenteel heb ik voor mijn huidige opdracht, WAFS, veel JavaScript gebruikt, maar de voorbeelden die Kilian gaf hebben mij geïnspireerd om in de toekomst minder JavaScript te gebruiken voor mijn website. Ook heeft Kilian mij gemotiveerd om meer aan de slag te gaan met Webdesign- en development, omdat hij zijn gastlezing op een hele interessante manier kon overbrengen, waardoor ik vanaf begin tot eind aandachtig naar hem kon luisteren!
